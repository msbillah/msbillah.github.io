About Me

  • Roadway feature mapping using 3D LiDAR
  • Centimeter-accurate realtime navigation using GPS/INS

Research

* Surface Reconstruction

Using 2D laser scanners (LiDAR) and GPS-INS to generate highly-accurate 3D surface: Currently involved in a project which uses 2D LiDARs and GPS-INS system to generate highly-accurate 3D surface.

* Point Cloud Feature Extraction

Roadway feature extraction from 3D point clouds using a graph based clustering approach: Developed a graph based clustering approach to extract roadway features directly from 3D point cloud.

* Roadway Feature Mapping

Roadway feature mapping using mobile laser scanner (LiDAR): Developed a process to extract roadway features and create map for intelligent driving assistance systems and future self-driving cars from 3D mobile laser scanner and GPS-INS data. Generated roadway map for 11 intersections in Palo-Alto testbed.
